وصف الدورة

تعد React.js واحدة من أكثر مكتبات JavaScript شيوعًا لبناء واجهات المستخدم التفاعلية. في هذه الدورة الشاملة، ستتعلم كيفية إنشاء تطبيقات ويب حديثة باستخدام React من الصفر وحتى مستوى متقدم.

سنبدأ بأساسيات React مثل المكونات والخصائص والحالة، ثم ننتقل إلى موضوعات أكثر تقدمًا مثل إدارة الحالة باستخدام Redux، والتوجيه، والتعامل مع واجهات برمجة التطبيقات (APIs)، وأفضل الممارسات في تطوير تطبيقات React.

ستقوم ببناء 5 مشاريع عملية خلال الدورة، بدءًا من تطبيقات بسيطة وصولاً إلى تطبيق متكامل يمكن استخدامه في محفظة أعمالك.

ماذا ستتعلم

  • أساسيات React.js والمفاهيم الرئيسية
  • إنشاء وتنظيم مكونات React
  • إدارة الحالة باستخدام Hooks
  • التوجيه باستخدام React Router
  • إدارة الحالة العالمية باستخدام Redux
  • التعامل مع واجهات برمجة التطبيقات (APIs)
  • اختبار تطبيقات React
  • نشر تطبيقات React على الإنترنت
